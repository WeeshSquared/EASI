EASI Programming Language Help File

The EASI Programming language was designed from the start to introduce programming to new audiences. Its English derived syntax allows beginners to focus on logic rather than learning mundane syntax. The following reference details the specifics of the EASI Programming Language.

Commands:

ASK

The ask command asks the user a question through a Popup box, then stores the answer inside a variable that is specified
Syntax: ASK "Write Question Here?" STORE TO exampleVariable

COMMENT or NOTE

The NOTE command allows for annotating of a program's source code, anything that follows the NOTE command is ignored by the EASI Programming Language.
Syntax: NOTE insert annotation here!
other Syntax: COMMENT insert annotation here!

CLEAR

The CLEAR command clears either the Console or Graphical Output screens.
Syntax (Console): CLEAR
Syntax (Graphical): CLEAR GRAPH

CREATE or DECLARE

The CREATE command creates a variable that can then be used afterwards in the program, the variable can optionally be set to a default value as well.
Syntax: CREATE variable_name
Syntax (set value): CREATE variable_name SET TO initial_value

DISPLAY or PRINT or SAY

The SAY command allows the program to display text on the Console Output Screen, the background and foreground colors can be specified.
Syntax (Default Colors): SAY "message"
Syntax (Change Foreground Color): SAY "message" in foreground_color
Syntax (Change Background and Foreground Colors): SAY "message" in foreground_color on background_color

DRAW

The DRAW command allows a program to draw a shape onto the Graphical Output Screen
Syntax (Rectangle): DRAW rectangle at (x_coordinate, y_coordinate) width by height
Syntax (Circle): DRAW circle at (x_coordinate, y_coordinate) radius radius_of_circle

END

The END command ends either a loop or a an if/if-else statement.
Syntax (Loop): END LOOP
Syntax (Game Loop): END GAME LOOP
Syntax (If): END IF

GAME LOOP

The GAME LOOP command allows the program to loop forever and maintain a high framerate when looping to allow for a game or animation to be written inside of the loop.
Syntax (Start): GAME LOOP
Syntax (END): END GAME LOOP

IF/ELSE IF/ELSE

The IF, ELSE IF, and ELSE statements allow for a condition to be checked.
Syntax (IF): IF condition
Syntax (ELSE IF): ELSE IF condition
Syntax (ELSE): ELSE
Example:
IF X > 5
  SAY "X is greater than 5"
ELSE IF X = 5
  SAY "X is equal to 5
ELSE
  SAY "X is less than 5
END IF

LOOP or REPEAT

The LOOP command allows a program to repeat a block of code a number of times, until a condition is true, or while a condition is true.
Syntax (Number of times): LOOP number_of_times COUNTER variable_that_stores_loop_count
Syntax (While condition is true): LOOP WHILE condition
Syntax (Until condition is true): LOOP UNTIL condition

SET

The SET command allows the program to set the value of a variable to a number or another variable or a mathematical expression containing variables and numbers.
Syntax: SET variable_name TO new_value

Reserved Variables

key_pressed

The key_pressed variable stores the value of the current keyboard key being pressed, it is especially useful when programming Video Games.
